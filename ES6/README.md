### ES6

> var、let、const

> > 区别一: 
- var 有变量提升的机制
- let 和 const 没有变量提升的机制

> > 区别二:
- var 可以重复声明同一个变量
- let 和 const 不能重复声明同一个变量
> > 区别三:
- var、let 是声明变量的
- const 是声明常量的，一旦声明必须赋值

> > 区别四:
- var、let 值是可以被修改的
- const 值一般情况下是不可以被修改的(除数组和对象以外)

> > 区别五:
- var 只有全局作用域和函数作用域，严格来说没有块级作用域
- let不仅有全局作用域和函数作用域还有块级作用域

> > 区别六:
- var和let都可以用来定义循环变量
- const不可以定义循环变量
